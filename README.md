# stocking-management-tool
概要： 本ツールは、店舗や倉庫における品出し・補充業務を効率化するためのWebベースの管理アプリケーションです。CSV形式のリストを読み込み、商品ごとの補充数（カウント）を直感的に管理できます。 主な機能： モード切替: 「飲料モード」と「酒モード」を切り替え、各リストを独立して管理可能。 リストの動的編集: CSVファイルからの読み込みに加え、ブラウザ上で直接商品の追加が可能。 補充数管理: 商品ごとに増減および個別リセット操作が可能。 検索機能: 大量のリストから対象商品を素早く検索し、該当箇所へ自動スクロール。 表示カスタマイズ: 文字サイズの変更に対応し、視認性を調整可能。 データ保持: localStorage を利用し、ブラウザを閉じても入力データが保持される。
