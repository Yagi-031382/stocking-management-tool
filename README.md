# 品出し管理ツール (Stocking Management Tool)

店舗や倉庫における品出し・補充業務を効率化するためのWebベースの管理アプリケーション。

## 概要
CSV形式で作成した商品リストを読み込み、現場で各商品の補充数をカウント・管理できるツール。
飲料モードと酒モードを切り替えて運用可能。

## 主な機能
- **モード切替**: 「飲料モード」と「酒モード」を切り替え、各リストを独立して管理。
- **データ永続化**: ブラウザのストレージを使用し、ページを閉じても入力内容を保持。
- **検索機能**: 商品名検索による自動スクロール。
- **柔軟なリスト編集**: CSVインポートに加え、ブラウザ上で直接商品を追加可能。
- **表示調整**: 現場の視認性に合わせた文字サイズ変更機能。

## 使い方
1. 本ツールをブラウザで開く。
2. 「飲料」または「酒」のモードを選択する。
3. CSVファイルをインポートするか、手動で商品を追加する。
4. 「＋」「－」ボタンで補充数をカウントする。
5. 作業完了後は「このモードを全リセット」または各商品の「0」ボタンで数値をクリアする。
